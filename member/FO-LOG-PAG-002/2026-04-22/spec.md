# FO-LOG-PAG-002 — 로그인 후 본인인증

## Description 1 — 본인인증 프로세스

### CI값 확인 분기

```
정상 로그인
└── CI값 확인
    ├── CI값이 빈값인 경우 → 대출 비교하기 (정상 흐름, 본인인증 불필요)
    └── CI값이 존재하는 경우 → 본인인증 진행 (1번)
```

### 1. 본인인증 화면

안내 문구:
```
대표자 정보가 변경되었습니다.
안전한 서비스 이용을 위해
본인인증을 진행해 주세요.
```

| 버튼 | 동작 |
|------|------|
| `[본인인증 하기]` | 본인인증 진행 |

```
본인인증 진행
├── 실패 (1-2)
│   문구: 본인인증에 실패하였습니다.
│         다시 한번 시도해 주세요.
└── 성공 → 2번 처리
```

### 2. 본인인증 성공 처리

성공 문구:
```
본인인증이 완료되었습니다.
이제 대표자 권한으로 서비스를 이용하실 수 있습니다.
```

| 버튼 | 동작 |
|------|------|
| `[확인]` | 비밀번호 변경 여부 확인 후 분기 |

### 비밀번호 변경 팝업 분기

```
[확인] 클릭
├── 비밀번호 변경 연장여부 'N' + 마지막 비밀번호 변경일 기준 90일 경과 시 → 2-1 팝업
├── 비밀번호 변경 연장여부 'Y' + 연장 승인일 기준 90일 경과 시 → 2-2 팝업
└── 해당 없음 → 마이페이지 이동
```

### 2-1. 비밀번호 변경 권고 팝업 (90일 경과)

문구:
```
비밀번호를 변경하신 지 90일이 경과되었습니다.
안전한 서비스 이용을 위해 비밀번호 변경을 권장드립니다.
```

| 버튼 | 동작 |
|------|------|
| `[90일 후 변경]` | 마이페이지 이동 |
| `[비밀번호 변경]` | 마이페이지 > 비밀번호 변경 화면 이동 |

### 2-2. 비밀번호 변경 강제 팝업 (180일 경과)

문구:
```
비밀번호를 변경하신 지 180일이 경과되었습니다.
안전한 서비스 이용을 위해 비밀번호 변경이 필요합니다.
```

| 버튼 | 동작 |
|------|------|
| `[비밀번호 변경]` | 마이페이지 > 비밀번호 변경 화면 이동 |

확인필요: [비밀번호 변경 화면 경로 — 마이페이지 내 정확한 화면 ID 불명확]
확인필요: [2-2 팝업에서 '변경' 없이 닫기 가능한지 여부 — 강제 여부 불명확]
