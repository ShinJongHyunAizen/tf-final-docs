# FO-MAG-PAG-005 — 회원 탈퇴

## Description 1 — 회원 탈퇴 사유 입력 (#2208)

### 1. 회원탈퇴 사유 선택

탈퇴 사유 목록:

| 사유 | 설명 |
|------|------|
| 대출 조건 및 비용 불만족 | 이자율, 한도 등에 대한 불만족 / 더 나은 조건의 대출상품 발견 |
| 서비스 품질 문제 | 플랫폼 기능, 고객 지원 등으로 서비스 이용 불편 |
| 보안 또는 개인정보 문제 | 매출 데이터 제공, 개인정보 보호 등에 대한 신뢰 부족 |
| 서비스 불필요 | 대출 필요 없음 / 개인적인 사유로 더 이상 서비스 이용 불필요 |
| 기타 | 입력 필드 노출 (플레이스홀더: `기타 탈퇴 사유를 입력해 주세요.(30자 이내)`) |

**'기타' 선택 시 탈퇴 사유 입력 필수 (유효성 검사)**

### 2. 회원 탈퇴 전 유의사항

유의사항 안내 문구:
```
회원 탈퇴 시 회원 정보와 서비스 이용 기록은 모두 삭제되며 삭제된 데이터는 복구할 수 없습니다.
다만 법령 또는 회사 내부 정책에 따라 보관해야 하는 정보는 탈퇴 후에도 일정 기간 보관됩니다.
자세한 사항은 개인정보 처리 방침에서 확인하실 수 있습니다.
회원 탈퇴 후 동일 사업자 번호로 재가입 하더라도 탈퇴 전 회원정보와 서비스 이용 기록은 복구되지 않습니다.
```

탈퇴 동의 체크박스 (필수):
```
탈퇴 유의사항을 확인하였으며 탈퇴에 동의합니다.
```

### 3. 회원 탈퇴 버튼

- **활성화 조건**: 탈퇴 사유 선택 & 유의사항 동의 체크박스 체크
- **(3-1) 기타 선택 후 사유 미입력 시 얼럿**: `기타 사유를 입력해 주세요.`
- 정상 입력 시 탈퇴 프로세스 진행

## Description 2 — 회원 탈퇴 로직 개선 (#5176)

### 회원 유형별 탈퇴 조건

| 회원 유형 | 탈퇴 조건 |
|---------|---------|
| 직원 | 예외조건 없이 탈퇴 가능 (대출/데이터 이용중 무관, 담당자 지위도 무관 #CCNLND-2726) |
| 대표 | 아래 탈퇴 가부 판단 플로우 적용 |

### 대표 탈퇴 가부 판단 플로우

```
탈퇴 시도 (대표)
│
├── 1. 진행중인 데이터제공동의 미확인 & 만료일 경과X
│   └── 탈퇴 불가 → 얼럿 [데이터 제공 거절 안내]
│       버튼: [데이터 제공 거절하기], [닫기]
│       [데이터 제공 거절하기] 클릭 →
│           비담보: FO-DEP-PAG-004
│           담보: FO-DEP-PAG-002
│
├── 2. 대출상태 == 신청완료
│   └── 탈퇴 불가 → 얼럿 [대출 신청 취소 안내]
│       버튼: [대출 신청 취소], [닫기]
│       [대출 신청 취소] 클릭 →
│           신청완료 1건: FO-LMG-MOD-001
│           신청완료 2건 이상: FO-LMG-PAG-001
│       [닫기] → 모달 닫힘
│
├── 3. 데이터 이용 상태 이용중 (대출 및 데이터제공)
│   └── 탈퇴 불가 → 얼럿 4 [데이터 및 대출 중지]
│       (판매몰 연동 0건 또는 비담보 데이터 제공동의인 경우 포함)
│       버튼: [데이터 및 대출 중지], [닫기]
│
├── 4. 현재 진행중인 데이터 이용건 있음 (대출 중지 조건)
│   └── 탈퇴 불가 → 얼럿 [데이터 제공 철회]
│       버튼: [데이터 제공 철회], [닫기]
│
├── 5. 등록 직원 있음 (회원 매핑된 직원)
│   └── 탈퇴 불가 → 직원 안내 얼럿
│       버튼: [직원관리 이동] → FO-EMP-PAG-001, [닫기]
│       탈퇴 시 직원 초대 딥링크 만료 처리
│       탈퇴 시 하위 직원 useyn = N 처리
│
└── 조건 없음 → 탈퇴 진행
    → 완료 얼럿: 회원탈퇴가 완료되었습니다. 그동안 이용해 주셔서 감사합니다.
    → 회원탈퇴 완료 & 메인 이동
```

### 얼럿 문구 정의

**1번 — 데이터 제공 거절 안내 얼럿**
```
{금융기관명}과 협약된 대출을 위한 데이터 제공 동의가 요청되어 있습니다.
데이터 동의 절차 진행중인 경우, 회원 탈퇴가 불가합니다.
데이터 제공동의 화면에서 [데이터 제공 거절하기] 후 탈퇴를 진행해 주세요.

데이터 제공을 거절 하시겠습니까?
```

**2번 — 대출 신청 취소 안내 얼럿**
```
현재 진행중인 대출 신청건이 있습니다.
대출 이용중에는 회원탈퇴를 할 수 없습니다.
대출관리 화면에서 [대출 신청 취소]후 탈퇴를 진행해 주세요.

대출을 취소 하시겠습니까?
```

**3번 — 데이터 및 대출 중지 얼럿**
```
진행 중인 데이터 및 대출이 있는 경우, 회원 탈퇴가 불가합니다.
진행 중인 데이터 및 대출건을 모두 중단하고 탈퇴하시겠습니까?
요청 시 담당자 확인 후 처리 완료 문자를 보내드립니다.
처리 완료 후 탈퇴를 다시 진행해 주세요.

이용중인 데이터 제공 {n}건
{금융기관명} | {상품명}

확인중인 대출 {n}건
{금융기관명} | {상품명}

이용중인 대출 {n}건
{금융기관명} | {상품명}
```
(여러 건인 경우 줄바꿈하여 1줄 1개씩 노출)

**4번 — 데이터 제공 철회 얼럿**
```
현재 진행중인 데이터 이용건이 있습니다.
{금융기관명}과 협약된 대출을 위한 데이터 제공 철회 시 탈퇴가 가능합니다.

데이터 제공을 철회 하시겠습니까?
```

**5번 — 직원 안내 얼럿**
```
사업자에 속한 직원이 있습니다.
직원을 사업자에서 내보내기 후 회원탈퇴를 진행해 주세요.

직원 관리 화면으로 이동하시겠습니까?
```

### 데이터 및 대출 중지 요청 처리 (3번 얼럿)

| 프로세스 타입 | 상태 | 처리 결과 |
|---|---|---|
| 비담보 | - | 처리상태 '신청', 요청 보 '지' 생성 |
| 담보 | 확인 | 처리상태 '신청', 요청 보 '신청 거절' 생성 |
| 담보 | 승인 | 처리상태 '신청', 요청 보 '종료' 생성 |
