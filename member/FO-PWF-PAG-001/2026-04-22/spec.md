# FO-PWF-PAG-001 — 비밀번호 찾기

## 공통 전제 — 인증 방식 변경

기존: 사업자번호 조회 → 휴대폰 본인인증
변경: 아이디 + 휴대폰 본인인증

---

## Description 1 — 비밀번호 찾기

### 3-1. 안내 문구
```
가입한 아이디를 입력해 주세요.
휴대폰 본인인증을 통해
비밀번호를 변경할 수 있습니다.
```

### 3-2. 아이디 입력 필드

| 필드 | 플레이스홀더 |
|------|------------|
| 아이디 | 아이디 |

### 3-3. 휴대폰 본인인증

| 버튼 | 동작 |
|------|------|
| `[휴대폰 본인인증]` | 아이디 + 휴대폰 본인인증 진행 |

```
본인인증 진행
├── 아이디 + 본인인증 성공 → FO-PWF-PAG-002 (비밀번호 재설정) 화면 이동
├── 실패1: CI값으로 가입된 이력이 없는 경우 → 3-4 팝업 노출
└── 실패2: 아이디와 본인인증 CI값이 일치하지 않는 경우 → 3-5 팝업 노출
```

### 3-4. 실패 팝업 — CI값으로 가입 이력 없음

문구:
```
본인 명의 휴대전화로 가입된 아이디를 찾을 수 없어요.
고객센터 문의 혹은 비밀번호 찾기를 다시 시도해 주세요.
```

| 버튼 | 동작 |
|------|------|
| `[확인]` | 팝업 닫힘, FO-PWF-PAG-001 화면 노출 |
| `[1:1 문의하기]` | FO-CSM-PAG-002 (1:1문의) 화면 이동 |

### 3-5. 실패 팝업 — 아이디 불일치

문구:
```
아이디가 일치하지 않아요.
고객센터 문의 혹은 비밀번호 찾기를 다시 시도해 주세요.
```

| 버튼 | 동작 |
|------|------|
| `[확인]` | 팝업 닫힘, FO-PWF-PAG-001 화면 노출 |
| `[1:1 문의하기]` | FO-CSM-PAG-002 (1:1문의) 화면 이동 |
