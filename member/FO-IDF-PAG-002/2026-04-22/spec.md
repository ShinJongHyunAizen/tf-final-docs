# FO-IDF-PAG-002 — 아이디 찾기 완료

## Description 1 — 아이디 찾기 완료

### 안내 문구
```
아이디를 찾았어요.
```

### 아이디 노출 규칙

```
회원 유형
├── 대표 (사업자 매핑됨)
│   ├── 아이디 표시
│   └── 사업자명 마스킹 처리
│       ├── 사업자명 글자수 7자 미만 → 앞 2자리 노출 + 나머지 글자수만큼 마스킹
│       └── 사업자명 글자수 7자 이상 → 앞 6자리 노출 + 나머지 글자수만큼 마스킹
└── 직원 (사업자 미매핑)
    └── 아이디만 노출 (사업자명 미노출)
```
