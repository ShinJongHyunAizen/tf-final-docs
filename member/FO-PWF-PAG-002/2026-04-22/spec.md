# FO-PWF-PAG-002 — 비밀번호 재설정

## Description 1 — 비밀번호 재설정

### 안내 문구
```
본인 확인되었습니다.
비밀번호 재설정을 해주세요.
```

### 비밀번호 입력 필드

| 필드 | 플레이스홀더 |
|------|------------|
| 비밀번호 | 비밀번호 |

- default: 입력 시 마스킹 처리, 비밀번호 뷰 off 아이콘
- 비밀번호 뷰 아이콘 클릭 시 → 입력한 비밀번호 노출, 뷰 on 아이콘으로 변경
- 비밀번호 뷰 on 아이콘 재클릭 시 → 비밀번호 마스킹 처리, 뷰 off 아이콘 노출

### 버튼

| 버튼 | 활성화 조건 | 동작 |
|------|-----------|------|
| `[비밀번호 재설정]` | 비밀번호 입력 시 활성화 | 비밀번호 재설정 진행 |

### 재설정 결과

```
비밀번호 재설정 진행
└── 성공 → 3-6 팝업 노출
    문구: 비밀번호가 재설정되었어요.
          로그인을 진행해 주세요.
    버튼: [로그인] → FO-LOG-PAG-001 화면 이동
```
