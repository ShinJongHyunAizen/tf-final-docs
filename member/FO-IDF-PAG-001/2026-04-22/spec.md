# FO-IDF-PAG-001 — 아이디 찾기

## 공통 전제 — 인증 방식 변경

기존: 사업자번호 조회 → 휴대폰 본인인증
변경: 휴대폰 본인인증으로 단순화

배경: 회원체계 변경 후 사업자가 없는 경우 존재 (#1526)
- 예) 사업장에서 초대한 회원이 내보내기 한 경우

---

## Description 1 — 아이디 찾기

### 1-1. 안내 문구
```
휴대폰 본인인증을 통해
아이디를 찾을 수 있습니다.
```

### 1-2. 휴대폰 본인인증

| 버튼 | 동작 |
|------|------|
| `[휴대폰 본인인증]` | 휴대폰 본인인증 진행 |

```
본인인증 진행
├── 성공 → FO-IDF-PAG-002 (아이디찾기 완료) 화면 이동
└── 실패 → 1-3. 아이디 찾기 실패 팝업 노출
```

### 1-3. 아이디 찾기 실패 팝업

문구:
```
본인 명의 휴대전화로 가입된 아이디를 찾을 수 없어요.
고객센터 문의 혹은 회원가입을 진행해 주세요.
```

| 버튼 | 동작 |
|------|------|
| `[1:1 문의하기]` | FO-CSM-PAG-002 (1:1문의) 화면 이동 |
| `[회원가입]` | FO-REG-PAG-001 (회원가입) 화면 이동 |
